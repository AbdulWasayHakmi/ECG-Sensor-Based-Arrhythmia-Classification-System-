# ECG-Sensor-Based-Arrhythmia-Classification-System-
Designed 3 lead ECG sensor using instrumentation Amplifier. Captured and imported the sensor data to LabView via  UART using Arduino. Performed FFT on the signal to get the heartbeat in LabView. Transmitted the signal data to  MATLAB to pass it through the Arrhythmia classification model via TCP/IP communication.
